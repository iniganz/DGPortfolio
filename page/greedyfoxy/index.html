About The Game

Greedy Foxy is a 2D platformer game where you play as a hungry fox navigating through challenging levels. Collect food, avoid obstacles, and reach the finish line to complete each stage!

Built with Godot Engine as a personal project to explore game development and level design concepts.

Tech Stack

  • 🎮
    Godot Engine 4 Game Engine
  • 📝
    GDScript Programming Language
  • 🎨
    Pixel Art Art Style
  • 🌐
    HTML5 Export Web Playable

Features
